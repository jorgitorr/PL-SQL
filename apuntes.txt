    DATE: este tipo de datos almacena una fecha y hora en formato estándar de Oracle. Para declarar una variable de tipo DATE, puedes usar la siguiente sintaxis:

    fecha DATE;
    Para asignar un valor a una variable de tipo DATE, puedes utilizar la función TO_DATE(), como se muestra en el siguiente ejemplo:


    fecha := TO_DATE('2023-04-26 15:30:00', 'YYYY-MM-DD HH24:MI:SS');
